#  Online Shoe Ordering System

A web-based Online Shoe Ordering System developed to provide customers with a convenient platform for browsing, ordering, and managing shoe purchases online. The system allows users to explore available products, add items to a shopping cart, place orders, and manage their accounts through a user-friendly interface.

The system also includes an administrator panel for managing products, customers, and orders efficiently.

---

##  Overview

The Online Shoe Ordering System is a full-stack web application developed using PHP and MySQL. This project demonstrates the implementation of an e-commerce platform with customer and administrator functionalities.

The system applies important web development concepts such as database management, CRUD operations, user authentication, session handling, and responsive web design.

---

##  Features

###  Customer Features

- User Registration and Login
- Browse Shoe Products
- View Product Details
- Add Products to Shopping Cart
- Manage Cart Items
- Place Orders
- View Order Information
- Manage User Profile

###  Administrator Features

- Admin Login
- Dashboard Overview
- Manage Shoe Products
- Add, Edit, and Delete Products
- Manage Customer Accounts
- Manage Orders
- Monitor Store Transactions
